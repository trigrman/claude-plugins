# Claude Code Plugins

Custom Claude Code plugin marketplace by Mike Mitchell.

## Usage

Add this marketplace:
```bash
/plugin marketplace add trigrman/claude-plugins
```

Install a plugin:
```bash
/plugin install granola-meetings@trigrman-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [granola-meetings](https://github.com/trigrman/granola-meetings) | Access and review Granola meeting notes |
